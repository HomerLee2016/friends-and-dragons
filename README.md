# Friends & Dragons

Small helper tools for the Friends and Dragons project.

## Overview

This repository contains lightweight utilities and examples used when working with the "Friends and Dragons" project. It's intentionally minimal — a tiny HTML demo (`thin-browser.html`) and supporting documentation to get started quickly.

## Features

- Tiny browser demo for rapid UI testing
- Simple, dependency-free files you can open locally

## Quick Start

1. Open the demo directly in your browser: [thin-browser.html](thin-browser.html)
2. Or serve the folder over a simple HTTP server (recommended for some browser features):

```bash
python -m http.server 8000
# then open http://localhost:8000/thin-browser.html
```

## Development

- Edit `thin-browser.html` directly and refresh the browser to see changes.
- Keep changes small and focused — this repo is intended for quick experiments.
